# grupo20_Chatbot
Repositorio del Proyecto: Chatbot Asistente en la Nube para Atención Automatizada (Lite) 
Proyecto académico que simula el diseño e implementación de un chatbot asistente en la nube, orientado a la atención automatizada de preguntas frecuentes y el registro de solicitudes básicas de CSDC.
El proyecto se desarrolla por fases, incluyendo análisis, diseño, implementación, pruebas y documentación final.
